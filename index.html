Position

Well ... position is totally screwed up ... no other words!

Tag Name Description
static Static Position This is the default value for position property. It is a normal flow of the document.
relative Relative Position Similar to static, but now we get the freedom to offset the position relative to itself.
absolute Absolute Position Takes the element out of the regular flow of the document.
fixed Fixed Position it behaves the same as absolute, but now with no space created for the element in the document layout.
sticky Sticky Position Here the element is positioned according to the document's normal flow, but is offset relative to its nearest scrolling parent.
